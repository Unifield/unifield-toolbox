- The instance name is now mandatory with the command 'run':
  example: ./runbot.py run jfb-uf-1347

- Better concurrency control, and free port detection

- Command deploy: new option "--unit":  ./runbot.py deploy -u &lt;jira-user&gt; &lt;UF-number&gt; --unit
  example: ./runbot.py deploy -u md 1129 --unit
